# SimplePySideSerial
基于PySide6的极简风格串口小助手

# 特性
- 布局复刻 Arduino 的串口助手。
- 更改串口，波特率，通讯格式
- 记忆用户配置
- 基于PySide6实现，提供UI布局文件

# 配置并运行
    # 安装需要的依赖
    pip install -r requirements.txt
    # 直接运行主程序
    python main.py
